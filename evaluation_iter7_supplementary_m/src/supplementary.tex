\section{Extended Methods}

\subsection{Null Model Construction}
The null models are generated using the Goni Method 1 edge-swap algorithm,
which preserves in-degree and out-degree sequences while randomizing connectivity.
The algorithm proceeds as follows:

\begin{enumerate}
\item Initialize with the original directed acyclic graph (DAG).
\item For each swap iteration (total swaps = $10 \times |E|$):
  \begin{enumerate}
  \item Select two random edges $(u,v)$ and $(x,y)$.
  \item Propose swap: replace with $(u,y)$ and $(x,v)$.
  \item Accept swap only if the resulting graph remains acyclic (verified via topological sort).
  \end{enumerate}
\item Output the rewired graph as a null model.
\end{enumerate}

This procedure ensures that each null model preserves the degree distribution while
randomizing higher-order structure, making it appropriate for testing motif enrichment.

\section{Corpus-Level Statistical Framework}

\subsection{Why Per-Graph BH-FDR Fails}
With 50 null models per graph, the minimum achievable $p$-value is $1/51 \approx 0.0323$. The Benjamini--Hochberg FDR threshold for rank-1 of 800 tests is $0.05/800 = 6.25 \times 10^{-5}$. This resolution gap makes BH-FDR survival mathematically impossible for most tests, regardless of the true effect size.

\subsection{Corpus-Level Tests}
The corpus-level alternative treats each graph's Z-score as one observation and applies standard tests across $n=200$ graphs:
\begin{itemize}
\item One-sample $t$-test: $t = 55.0$, $p = 2.38 \times 10^{-122}$
\item Wilcoxon signed-rank: $W = 0$, $p = 1.44 \times 10^{-34}$
\item Sign test: 200/200 positive, $p = 1.24 \times 10^{-60}$
\item Cohen's $d = 3.89$ (large effect)
\end{itemize}

\subsection{Mixed-Effects Model}
We fit a random-intercept model: $Z_{ij} \sim \beta_0 + u_j + \varepsilon_{ij}$, where $j$ indexes domains.
\begin{itemize}
\item $\beta_0 = 47.2$, 95\% CI: [40.5, 53.8]
\item $p = 4.93 \times 10^{-44}$
\item $\mathrm{Var}(u_j) = 89.23$, $\mathrm{Var}(\varepsilon) = 67.36$
\item ICC $= 0.570$
\end{itemize}

\subsection{Deep Null FDR}
With 200 nulls per graph on 15 stratified graphs, 15/60 tests survive BH-FDR at $\alpha = 0.05$.

\subsection{Convergence}
Median convergence at $N = 30$ null models. 10\% converged by $N=5$, 70\% by $N=30$, 100\% by $N=50$.

\section{Limitations and Future Directions}

\subsection{Critical Limitations}
\begin{enumerate}[leftmargin=*]
\item \textbf{Single model (gemma-2-2b) — all findings may be model-specific} \\\textcolor{criticalred}{[critical]} Mitigation: acknowledged only. \\\textit{Future work: Extend to multiple architectures (GPT, Llama, Mistral) and model sizes (7B, 13B, 70B)}
\item \textbf{Attribution graph faithfulness — unknown whether graphs capture real model computation} \\\textcolor{criticalred}{[critical]} Mitigation: acknowledged only. \\\textit{Future work: Compare attribution methods; validate with ground-truth circuits from toy models}
\end{enumerate}

\subsection{Major Limitations}
\begin{enumerate}[leftmargin=*]
\item \textbf{No model-level causal intervention — graph-theoretic ablation does not prove functional importance} \\\textcolor{majororange}{[major]} Mitigation: partially addressed. \\\textit{Future work: Implement actual model-level ablation of FFL hub nodes and measure task accuracy change}
\item \textbf{3-node motif spectrum degeneracy — effectively 1 dimension, limiting diversity claims} \\\textcolor{majororange}{[major]} Mitigation: partially addressed. \\\textit{Future work: Extend to larger motifs (5-node+); develop weighted motif spectrum theory}
\item \textbf{Small unique motif R-squared (0.018) — most variance shared with graph statistics} \\\textcolor{majororange}{[major]} Mitigation: fully addressed. \\\textit{Future work: Investigate non-linear unique information measures; develop less-correlated motif features}
\item \textbf{BH-FDR eliminates per-graph significance — overrepresentation is corpus-level only} \\\textcolor{majororange}{[major]} Mitigation: fully addressed. \\\textit{Future work: Increase null models per graph (100+) to improve per-graph p-value resolution}
\item \textbf{Failure prediction completely negative — motifs cannot predict model errors} \\\textcolor{majororange}{[major]} Mitigation: fully addressed. \\\textit{Future work: Investigate whether finer-grained failure modes are predictable}
\end{enumerate}

\subsection{Minor Limitations}
\begin{enumerate}[leftmargin=*]
\item \textbf{Small per-domain sample sizes (\textasciitilde{}25 graphs per domain)} \\\textcolor{minorgreen}{[minor]} Mitigation: partially addressed. \\\textit{Future work: Collect more attribution graphs per domain; include more capability domains}
\item \textbf{Weak semantic consistency for 3-node FFLs (V=0.13)} \\\textcolor{minorgreen}{[minor]} Mitigation: partially addressed. \\\textit{Future work: Develop richer semantic characterization methods}
\item \textbf{Pruning sensitivity — Z-scores depend on pruning threshold (rho=0.29-0.50)} \\\textcolor{minorgreen}{[minor]} Mitigation: fully addressed. \\\textit{Future work: Develop pruning-invariant motif measures; use multiple thresholds in reporting}
\end{enumerate}
